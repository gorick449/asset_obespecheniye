## 1. Карточка оценки
- Номер строки: 47
- Название: Учёт РНПТ — автоматическая проверка РНПТ через веб-сервис ФНС
- Краткая формулировка требования: Должна поддерживаться проверка регистрационных номеров партии (РНПТ) через сервис ФНС с фиксацией результата и возможностью контроля в документах.
- Дата оценки: 2026-03-23
- Версия оценки: 1.0
- Цель бизнеса: Снизить риск ошибок в РНПТ до отражения операций и отчётности.
- Границы доработки (scope / out-of-scope): В scope: использование типового сервиса проверки, журнал статусов, отчёт контроля. Out-of-scope: изменение API ФНС, доработки на стороне оператора ЭДО.
