# Оценка: [строка 59] Анализ ТРУ за 365 дней

## 1. Карточка оценки
- **Номер строки:** 59
- **Название:** Анализ ТРУ за 365 дней (контрагент, договор, документ, счёт учёта, группировка по пунктам перечня ТРУ)
- **Дата оценки:** 23.03.2026
- **Версия оценки:** v1
- **Цель:** годовая аналитика движений с корпоративной классификацией ТРУ и трассировкой.
- **Границы:** отчёт СКД; обеспечение НСИ перечня ТРУ (10) и ID потребности (25) — в смежных задачах.

## 2. Статус реализации
- **Полностью реализовано:** нет
- **Частично реализовано:** да — типовые и кастомные отчёты по движению ТМЦ (**лтхОтчетПоДвижениюТМЦ** с расширенным описанием в справке; **ДвижениеСерииНоменклатуры** и др.); группировка именно по **пунктам перечня ТРУ** и полный набор разрезов из формулировки 59 по метаданным отчёта не подтверждены без просмотра схемы СКД.
- **Пояснение:** высокая вероятность доработки СКД или нового отчёта на базе регистров движений/бухгалтерии.

## 3. Подтверждённые факты
- Отчёт **лтхОтчетПоДвижениюТМЦ** (синоним включает «Отчет по движению ТМЦ (карточка ТМЦ)») предназначен для расширенного анализа движений (справка метаданных).
- Имеются отчёты и регистры для движений номенклатуры в различных разрезах (см. семантический поиск по движению номенклатуры).

## 4. Пробелы
- Наличие реквизита связи номенклатуры с **пунктом перечня ТРУ** (10).
- Производительность запроса за 365 дней без материализации.

## 5. Новые объекты
- Опционально: регламентное задание формирования агрегатов по дням/месяцам.

## 6. Существующие объекты
- **лтхОтчетПоДвижениюТМЦ**, регистры склад/бухгалтерия, справочник **Номенклатура**.

## 7. Зависимости
- 10, 25, 37, 38.

## 8. Роли

## 9. Варианты

2. **Архитектурный:** отдельная витрина оборотов с ночным расчётом + лёгкий отчёт.
