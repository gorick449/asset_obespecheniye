# Оценка: [строка 43] Планирование потребности в СИЗ

## 1. Карточка оценки
- Номер строки: 43
- Название: Планирование и учёт СИЗ — планирование потребности
- Краткая формулировка требования: план закупки СИЗ в разрезе номенклатуры и характеристик по статистике потребления, по нормам, с учётом резервов.
- Дата оценки: 23.03.2026
- Версия оценки: 1.0
- Цель бизнеса: свести в один управленческий цикл расчёт потребности для закупки СИЗ.
- Границы доработки (scope / out-of-scope): связь с общим планом/программой закупок (строка 24) и нормативами запасов (строка 11); не дублировать полный производственный MRP.

## 2. Статус реализации
- Полностью реализовано: нет — отдельного объекта «план закупки СИЗ» с явным объединением трёх источников в метаданных не обнаружено.
- Частично реализовано: да
- Не реализовано: нет (исходные данные и типовые механизмы обеспечения/закупок присутствуют).
- Не определено (требует проверки): наличие кастомной обработки `лтх*` с названием про план/расчёт, не попавшей в выборку по описанию.
- Вероятно покрыто частично: нормы — `лтхСпецодеждаДляДолжностей`; статистика/резервы — типовые «Обеспечение потребностей», «Настройка поддержания запасов», заказы по потребностям (подтверждено метаданными как подсистемы закупок/склада).
- Требует проверки: автоматическое сопоставление штатной численности с нормами и выгрузка в строку ПЗ без ручного Excel.
- Пояснение статуса (1-3 предложения): семантический поиск по конфигурации выявил контур спецодежды и общие механизмы планирования закупок ERP, но не выделил единый документ/обработку «план СИЗ» с тремя источниками в названии/описании. Интеграция с ПЗ остаётся зоной доработки или настройки сценариев.

## 3. Подтвержденные факты
- Справочник норм `лтхСпецодеждаДляДолжностей` (подтверждено метаданными) как источник «по нормам».
- Обработка «Обеспечение потребностей» (синоним: формирование заказов по потребностям) с шагами расчёта и поддержанием запасов (подтверждено текстом справки в метаданных).
- Обработка «Настройка поддержания запасов» с методами min-max, статистика, норма потребления (подтверждено метаданными).
- Общая команда «Формирование заказов на закупку по потребностям» (подтверждено метаданными).
- Регистр накопления `лтхСпецодеждаВыданнаяФизЛицам` с измерениями номенклатура и характеристика (подтверждено метаданными) — потенциальная база для статистики выдачи.
