# Оценка: [строка 28] Формирование лотов

## 1. Карточка оценки
- Номер строки: 28
- Название: Формирование лотов
- Краткая формулировка: подготовка лотов из строк плана закупок с правилами объединения/разбиения и последующей передачей в ИСУ Тендер.
- Дата оценки: 23.03.2026
- Версия: 1.0
- Цель бизнеса: структурирование закупочной процедуры по лотам с учётом 223-ФЗ и коммерческих сценариев.
- Границы: справочник/документ «Лот», закупочные процедуры, регистр «Лоты плана закупок»; out-of-scope — настройка внешней ИСУ Тендер.

## 2. Статус реализации
- Полностью реализовано: контур лотов в составе закупочной процедуры **подтверждён** справками к «Лот», «Закупочные процедуры», регистру «Лоты плана закупок».
- Частично реализовано: для 223-ФЗ **одна строка плана — один лот** (ограничение разбиения описано в справке); для коммерции — подбор из заявок/вручную.
- Пояснение: подтверждено **метаданными**; готовый коннектор к конкретной ИСУ Тендер (требование 61) в метаданных **не детализирован** в данной оценке.

## 3. Подтверждённые факты
- Справочник «Лоты» и документ «Лот» (в составе процедуры) описывают многолотовую модель, виды закупки (лотовая/попозиционная/зонтичная).
- Регистр сведений «Лоты плана закупок» связывает план и лоты.
- Закупочная процедура содержит расписание стадий и мероприятия (планирование сроков подготовки).

## 4. Пробелы
- Формат обмена с ИСУ Тендер — см. требование 61; проверить наличие HTTP/обработок с таким именем в прикладной базе.

## 5. Новые объекты
- При интеграции: точки обмена, сопоставление номенклатуры лота.

## 6. Существующие объекты
