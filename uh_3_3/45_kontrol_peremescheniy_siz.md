# Оценка: [строка 45] Контроль перемещения СИЗ, возврата на склад, списания

## 1. Карточка оценки
- **Номер строки:** 45
- **Название:** Контроль перемещения СИЗ, возврата на склад, списание
- **Краткая формулировка требования:** отслеживать перемещения выданных СИЗ, возврат на склад, списание из эксплуатации в связке с остатками и отчётом 53.
- **Дата оценки:** 23.03.2025
- **Версия оценки:** 1.0
- **Цель бизнеса:** исключить «потерю» выданных СИЗ, корректно закрывать сроки носки и списание.
- **Границы:** учёт в рамках конфигурации; отдельный физический склад «только СИЗ» — по необходимости заказчика.

## 2. Статус реализации
- **Полностью реализовано:** в типовом контуре спецодежды/инвентаря — **операции возврата из эксплуатации, списания из эксплуатации, перемещения между сотрудниками** описаны в справках документов.
- **Частично реализовано:** отдельный выделенный склад/группа «СИЗ» как обязательный механизм — **не подтверждён** как требование платформы; достигается настройкой складов и счетов.
- **Пояснение:** документ **Перемещение товаров, материалов** поддерживает вид операции **передача между сотрудниками** для материалов в эксплуатации (спецодежда и инвентарь) с заполнением по остаткам выданного сотруднику — подтверждено справкой объекта.

## 3. Подтверждённые факты
- **Возврат материалов из эксплуатации** — для спецодежды и спецоснастки; ссылка на партию передачи; заполнение по остаткам — подтверждено справкой документа.
- **Списание материалов из эксплуатации** — закладки по видам материалов, акт **МБ-8** — подтверждено справкой документа.
- **Перемещение товаров, материалов** — вид **передача между сотрудниками**, реквизиты **От кого / Кому**, заполнение по остаткам выданного — подтверждено справкой документа.
- Связь с требованием 44: на основании **Передача материалов в эксплуатацию** по справке можно вводить возврат и списание — подтверждено справками документов.

## 4. Открытые вопросы / пробелы
- Нужен ли заказчику **обязательный** отдельный склад СИЗ или достаточно аналитики по сотруднику/партии/счету МЦ?
- Требуются ли **блокирующие** контроли (запрет списания без актов) сверх типовых — не проверялось телами модулей в рамках данной оценки.

## 5. Новые объекты системы
- При необходимости отдельного склада: только элементы НСИ (**Склады**), без программной доработки.
- При необходимости жёсткого контроля маршрутов: возможны доработки модулей документов (оценка в варианте 2).

## 6. Существующие объекты
- **ПеремещениеТоваров**, **ВозвратМатериаловИзЭксплуатации**, **СписаниеМатериаловИзЭксплуатации**, **ПередачаМатериаловВЭксплуатацию**.

## 7. Зависимости
- **Внутренние:** 44 (выдача), 53 (отчёт), 38–39 (склад/перемещения запасов — для пересечения с «обычными» ТМЦ).

## 8. Ролевая модель
