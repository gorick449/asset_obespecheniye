# Оценка: [строка 26] Согласование плана/программы закупок

## 1. Карточка оценки
- Номер строки: 26
- Название: Согласование плана/программы закупок
- Краткая формулировка: маршрут утверждения программы/строк плана и влияние на допустимость последующих этапов (лоты, тендеры, заказы).
- Дата оценки: 23.03.2026
- Версия оценки: 1.0
- Цель бизнеса: управляемый переход от черновика к утверждённому плану с контролем полномочий.
- Границы: объекты «Программа закупок», «Строка плана закупок», механизмы согласования УХ; out-of-scope — внешнее согласование в ИСУ без ТЗ.

## 2. Статус реализации
- Полностью реализовано: **не зафиксировано кодом** в данной сессии; в справке к документу «Программа закупок» указано подключение к согласованию и набор статусов (черновик, на утверждении, утверждён, ошибки/выгрузка/размещение для 223-ФЗ).
- Частично реализовано: статусная модель программы и наследование статусов строк после утверждения (описано в справке метаданных).
- Пояснение: подтверждено **метаданными (справка)**; точка входа маршрута и серверные проверки **требуют** точечного BSL-обзора при необходимости статуса «полностью».

## 3. Подтверждённые факты
- Документ «Программа закупок» содержит реквизиты периода, организации, даты утверждения, версии, УИД ЕИС — интегрирован в контур 223-ФЗ.
- В справке к «Программа закупок» явно указаны статусы и связь строк плана с программой после утверждения.

## 4. Пробелы
- Явные команды смены статуса и блокировки лотирования до утверждения — проверить в модулях объектов и форм.

## 5. Новые объекты
