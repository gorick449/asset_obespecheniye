# Оценка: [строка 31] Формирование тендерной документации по шаблонам

## 1. Карточка оценки
- Номер строки: 31
- Название: Формирование тендерной документации по шаблонам
- Краткая формулировка: подготовка комплекта документации из данных лота/плана по шаблонам (печать/Office).
- Дата оценки: 23.03.2026
- Версия: 1.0
- Цель бизнеса: ускорение подготовки КД и снижение ошибок заполнения.
- Границы: макеты печатных форм, выгрузки пакетов, присоединённые файлы; out-of-scope — юридическая экспертиза комплекта.

## 2. Статус реализации
- Полностью реализовано: **базовый механизм** выгрузки пакета закупочной документации в файл (архив) для загрузки на ЭТП описан в справке к «Закупочные процедуры».
- Частично реализовано: **полный набор шаблонов под регламент заказчика** и автозаполнение всех видов документов **не инвентаризированы** в данной оценке (нет полного списка макетов по лоту в выборке).
- Пояснение: подтверждено **справкой метаданных** закупочной процедуры; детальный перечень макетов — пробел.

## 3. Подтверждённые факты
- В справке к «Закупочные процедуры» указана возможность выгрузить пакет документации в архив для ЭТП.
- Используются шаблоны заполнения реквизитов и настройки процесса выбора (упоминание шаблонов в справке).
- Отчёт/печатные формы протоколов присутствуют в подсистеме (журнал протоколов закупочных процедур).

## 4. Пробелы
- Сопоставление требуемых «типов документов» заказчика с фактическими макетами в конфигурации.

## 5. Новые объекты
- При нехватке макетов: новые макеты печатных форм, СКД, обработки генерации docx.

## 6. Существующие объекты
- «Закупочные процедуры», «Лот», присоединённые файлы, отчёт «Протокол закупочной процедуры» (упоминание в семантическом поиске по победителям).

## 7. Зависимости
