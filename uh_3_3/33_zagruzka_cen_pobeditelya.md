# Оценка: [строка 33] Загрузка цен победителя в справочник номенклатуры

## 1. Карточка оценки
- Номер строки: 33
- Название: Загрузка цен победителя в справочник номенклатуры
- Краткая формулировка: перенос цен из предложения/протокола в регистр цен номенклатуры.
- Дата оценки: 23.03.2026
- Версия: 1.0
- Цель бизнеса: актуальные цены для планирования и договоров.
- Границы: документ «Установка цен номенклатуры», РС «Цены номенклатуры», связь с предложением; out-of-scope — пересчёт всех прайсов поставщика.

## 2. Статус реализации
- Полностью реализовано: в конфигурации присутствуют **документ «Установка цен номенклатуры»** и **регистр сведений «Цены номенклатуры»** (подтверждено семантическим поиском метаданных).
- Частично реализовано: **автоматическое заполнение установки цен из протокола/предложения** как единый готовый сценарий **не подтверждён** без чтения модулей проведения/обработчиков.
- Пояснение: подтверждено **наличием объектов**; поведение «одной кнопкой из победителя» — гипотеза до BSL.

## 3. Подтверждённые факты
- Объекты для фиксации цен номенклатуры существуют (документ + РС).
- В «Предложение поставщика» хранятся цены по строкам номенклатуры.

## 4. Пробелы

## 5. Новые объекты

## 6. Существующие объекты
