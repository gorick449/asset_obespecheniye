# Оценка: [строка 58] Сравнительный анализ цен

## 1. Карточка оценки
- Номер строки: 58
- Название: Сравнительный анализ цен
- Краткая формулировка требования: источники цен (предложения, победитель, поступления, внешние); глубина истории и разрезы; устойчивость при слабом учёте предложений [строка 32].
- Дата оценки: 23.03.2026
- Версия оценки: 1
- Цель бизнеса: обоснование выбора поставщика и контроль рыночного уровня цен.
- Границы доработки (scope / out-of-scope): отчётность и использование существующих документов КП/протоколов; не замена модуля ценообразования.

## 2. Статус реализации
- Полностью реализовано: не подтверждено.
- Частично реализовано: высокая вероятность — отчёт **СравнениеПредложенийПоставщиков**; в коде есть **СравнениеПоступившихПредложений** у документа **ЗапросКоммерческихПредложенийПоставщиков** и логика победителя в **ПротоколВыбораПобедителей**.
- Не реализовано: единый отчёт с всеми внешними прайсами и «последней закупкой» [строка 35] в одной схеме — не подтверждено.
- Требует проверки: заполнение при отсутствии активного сбора КП [строка 32].
- Пояснение статуса (1-3 предложения): Подтверждены объекты сравнения предложений и серверная процедура сравнения по запросу КП. Расширение до полного «рынка» с внешними источниками [строка 34] и устойчивые разрезы истории — зона доработки.

## 3. Подтверждённые факты
- Отчёт **СравнениеПредложенийПоставщиков** существует в категории «Отчеты» — подтверждено анализом метаданных.
- Процедура **СравнениеПоступившихПредложений** в менеджере документа **ЗапросКоммерческихПредложенийПоставщиков** предназначена для сравнения выбранных коммерческих предложений — подтверждено BSL (поиск по описанию).
- В менеджере **ПротоколВыбораПобедителей** есть функции проверки предложения победителя (**ЭтоПредложениеПобедителя** и связанные) — подтверждено BSL.

## 4. Открытые вопросы / пробелы (если есть)
- Включение цен фактических поступлений и внешних прайсов в одну схему.
- Период истории 12+ месяцев и производительность.

## 5. Новые объекты системы

## 6. Существующие объекты, требующие доработки
