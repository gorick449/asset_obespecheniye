# Оценка: [строка 49] Контроль остатков по РНПТ в разрезе партий

## 1. Карточка оценки
- **Номер строки:** 49
- **Название:** Остатки по РНПТ в разрезе партий
- **Краткая формулировка:** учёт количества прослеживаемости по РНПТ и согласованность с количественными запасами; основа для уведомлений (50).
- **Дата оценки:** 23.03.2025
- **Версия оценки:** 1.0
- **Цель бизнеса:** исключить отрицательные/необоснованные остатки по прослеживаемости.
- **Границы:** типовые регистры и отчёты прослеживаемости; не включает нестандартные партии вне конфигурации.

## 2. Статус реализации
- **Полностью реализовано:** **на уровне метаданных** присутствует регистр накопления **ПрослеживаемыеТовары** с измерением **РНПТ** и ресурсами **Количество**, **КоличествоПрослеживаемости** — подтверждено `object_structure`.
- Дополнительно в метаданных есть **ПрослеживаемыеТоварыПоОКПД2** — подтверждено семантическим поиском (для отдельных сценариев).
- **Согласованность с количественными запасами** как бизнес-контроль — требует настройки методики и периодических сверок; автоматический отчёт «идеальной» сверки в рамках данной проверки **не детализировался**.

## 3. Подтверждённые факты
- Структура регистра **ПрослеживаемыеТовары**: измерения включают **Организация**, **Номенклатура**, **РНПТ**, **ВидЗапасов** и др.; ресурсы — **Количество**, **КоличествоПрослеживаемости** — подтверждено метаданными.
- Регистр сведений **ОперацииСПрослеживаемымиТоварами** присутствует — подтверждено метаданными (назначение — в составе подсистемы прослеживаемости).

## 4. Пробелы
- Конкретный «главный» отчёт для заказчика по остаткам РНПТ (имя и форма) — не сопоставлялся с пользовательским ТЗ; в конфигурации есть отчётность по прослеживаемости (в т.ч. регламентированные формы уведомлений).

## 5. Новые объекты
- При необходимости **кастомного** контрольного отчёта — отчёт СКД (вариант 2).

## 6. Существующие объекты
- **ПрослеживаемыеТовары**, документы движения прослеживаемости, типовые отчёты подсистемы.

## 7. Зависимости
- **Внутренние:** корректное проведение поступлений/реализаций с РНПТ (38, 46–48).

## 8. Роли

## 9. Варианты реализации

2. **Специализированный отчёт контроля** расхождений между **ТоварыНаСкладах** (или аналогом) и **ПрослеживаемыеТовары**.
