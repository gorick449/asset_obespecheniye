# Строка 14 — допущения, ограничения, риски

## Допущения
- Конфигурация: Управление холдингом 3.3 (проиндексированная копия).
- «Кампания» может быть закрыта связкой Оперативный план + Периоды + параметры закупки потребностей.

## Ограничения анализа
- Не выполнялся полный прогон сценариев в продуктивной ИБ (даты блокировок, константы).
- Не перечислялись все регламентные задания контура.

## Риски
- Терминологический разрыв ТЗ ↔ объекты 1С.
- Регрессия лимитов и оперативного планирования при доработках.

## Варианты реализации (кратко)
- Донастройка периодов/проверок без нового справочника.
- Ввод справочника/документа «Кампания» с подписками и отчётностью.

## Чего не хватало для статуса «полностью реализовано»
- Явная сущность с тремя окнами и доказанными блокировками на уровне BSL для всех точек ввода.
