# Оценка: [строка 32] Сбор и анализ предложений поставщиков

## 1. Карточка оценки
- Номер строки: 32
- Название: Сбор и анализ предложений поставщиков
- Краткая формулировка: регистрация предложений, сравнение по критериям, подготовка к протоколу.
- Дата оценки: 23.03.2026
- Версия: 1.0
- Цель бизнеса: прозрачное сравнение условий и обоснованный выбор.
- Границы: документ «Предложение поставщика», протокол выбора, экспертная оценка; out-of-scope — внешняя оценка на ЭТП.

## 2. Статус реализации
- Полностью реализовано: документ «Предложение поставщика» с детализацией по номенклатуре, ценам, НДС, валюте; «Протокол выбора победителей» с ветками решений и переторжкой — в справке.
- Частично реализовано: **автоматический импорт предложений с ЭТП** не подтверждён как универсальный (в справке к процедурам указано отсутствие преднастроенной интеграции с ЭТП).
- Пояснение: подтверждено **метаданными и справкой**; деталь BSL по всем сценариям импорта — не полностью пройден.

## 3. Подтверждённые факты
- «Предложение поставщика» содержит шапку с контрагентом, валютой, датой и ТЧ условий по номенклатуре.
- «Протокол выбора победителей» поддерживает оценку предложений и переторжку (справка).
- В коде есть функции получения описания предложений победителей (модуль интеграции ЦУЗ–ЭТП).

## 4. Пробелы
- Наличие готового импорта для конкретной площадки заказчика.

## 5. Новые объекты

## 6. Существующие объекты
