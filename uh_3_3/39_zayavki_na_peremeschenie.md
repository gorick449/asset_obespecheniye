# Оценка: [строка 39] Заявки на перемещение (со склада)

## 1. Карточка оценки
- **Номер строки:** 39  
- **Название:** Формирование заявок на перемещение со склада  
- **Краткая формулировка требования:** Документ заявки при способе обеспечения «перемещение», резервирование и последующее перемещение; стадии для отчёта по потребности.  
- **Дата оценки:** 23.03.2026  
- **Версия оценки:** 1.0  
- **Цель бизнеса:** Оформить внутреннее перемещение как процесс с заявкой и контролем стадий.  
- **Границы:** *Scope* — заявка, резерв, связь с «ПеремещениеТоваров», отчётность. *Out* — внешняя WMS.

## 2. Статус реализации
- **Полностью реализовано:** нет  
- **Частично реализовано:** да  
- **Не реализовано:** нет (отдельный документ «заявка на перемещение» с ожидаемым именем не найден)  
- **Не определено (требует проверки):** да — фактический документ-заявка под другим именем  
- **Вероятно покрыто частично:** да — документ «ПеремещениеТоваров»  
- **Требует проверки:** механизм резервирования под перемещение в контуре обеспечения  
- **Пояснение:** По метаданным среди документов с именем, содержащим «ЗаявкаНаПеремещ», совпадений нет. Документ «ПеремещениеТоваров» существует (склады-отправитель/получатель, виды операций, ТЧ «Товары») — подтверждено метаданными и справкой. Для требуемой «заявки» вероятны: использование иного документа заказа/внутреннего заказа или необходимость разработки.

## 3. Подтверждённые факты
- Документ «ПеремещениеТоваров»: реквизиты складов отправителя/получателя, подразделения, виды операций; ТЧ для товаров — подтверждено метаданными.  
- В справке к «ПеремещениеТоваров» указана возможность ввода на основании «Поступление (акт, накладная)» — подтверждено текстом справки.  
- Поиск документа по шаблону имени «ЗаявкаНаПеремещ*» в категории документов дал пустой результат — подтверждено метаданными.

## 4. Открытые вопросы / пробелы
- Является ли «внутренний заказ» или иной документ фактической заявкой.  
- Нужны ли статусы заявки отдельно от статуса перемещения.  
- Правила резерва: по назначению, по заказу на перемещение.

## 5. Новые объекты системы
- При отсутствии подходящего документа: «ЗаявкаНаПеремещениеТоваров» (или аналог) + регистр резервов/статусов.

## 6. Существующие объекты, требующие доработки
