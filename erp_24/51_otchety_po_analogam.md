# Оценка: [строка 51] Отчёты по аналогам (частота, эффективность)

## 1. Карточка оценки
- Номер строки: 51
- Название: Отчёты по аналогам
- Краткая формулировка: аналитика использования аналогов (частота, эффективность) на основе учёта в документах.
- Дата оценки: 23.03.2025
- Версия оценки: 1.0
- Цель бизнеса: управление заменами номенклатуры в закупках и оценка выгоды/рисков.
- Границы: отчёт СКД, источники движений/журналов; вне scope — ML-прогнозы.

## 2. Статус реализации
- Полностью реализовано: **нет**
- Частично реализовано: **не подтверждено** (нет явного отчёта по ключевым словам)
- Не реализовано: **да** — в категории отчётов не найден объект с семантикой «аналоги номенклатуры»; поиск по имени «Аналог» в отчётах дал нерелевантные совпадения
- Пояснение: Без подтверждённого учёта подбора аналогов в документах ([строка 13]) отчёт будет пустым даже после разработки.

## 3. Подтвержденные факты
