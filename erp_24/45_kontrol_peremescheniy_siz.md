# Оценка: [строка 45] Контроль перемещения СИЗ, возврата, списания

## 1. Карточка оценки
- Номер строки: 45
- Название: Контроль перемещения СИЗ, возврата на склад, списания
- Краткая формулировка: замкнутый цикл движения СИЗ с различением склада/группы и влиянием на остатки и отчёт 53.
- Дата оценки: 23.03.2025
- Версия оценки: 1.0
- Цель бизнеса: исключить «потерю» выданного СИЗ и обеспечить корректные остатки для аналитики.
- Границы: документы и регистры 1С; вне scope — физический контроль на проходной.

## 2. Статус реализации
- Полностью реализовано: нет (в части **специализированного** контура СИЗ)
- Частично реализовано: да — общий складской/ОС контур может покрывать операции без маркировки «СИЗ»
- Не реализовано: не выявлены кастомные документы `лэс_*` для возврата/перемещения/списания именно СИЗ (в выборке документов с префиксом лэс по фильтру СИЗ/спец — только заявка на выдачу)
- Не определено: используются ли типовые `ПеремещениеТоваров`, `ВнутреннееПотребление`, `СписаниеНедостач` с номенклатурной группой СИЗ
- Пояснение: Учёт выдачи подтверждён ([строка 44]); обратные операции как отдельный предметный контур в метаданных с префиксом заказчика **не подтверждены**.

## 3. Подтвержденные факты
- Кастомный контур выдачи: `ЛЭС_ЗаявкаНаВыдачуСпецодежды`, `лэс_РаздаточнаяВедомостьСИЗ`, регистр `лэс_СпецодеждаДанныеДляЛичнойКарточки`.
- В коде `ВнутреннееПотреблениеТоваров` присутствуют обработчики/связи со спецодеждой (`ВыбратьСпецодежду`, `лэс_СпецодеждаПриИзменении`) — интеграция с типовым документом возможна.
- Подсистема «Спецодежда/СИЗ» присутствует в конфигурации.

## 4. Открытые вопросы / пробелы
- Какой документ оформляет возврат изношенного/невостребованного СИЗ?
- Выделен ли склад/группа номенклатуры СИЗ для запретов списания «мимо» выдачи?

## 5. Новые объекты системы
- При необходимости жёсткой связи с выдачей: документ «Возврат СИЗ» / «Списание СИЗ» с ссылкой на строку выдачи или регистр.

## 6. Существующие объекты, требующие доработки

- Возможно: расширение регистра `лэс_СпецодеждаДанныеДляЛичнойКарточки` для фиксации возврата/списания.
