# Оценка: [строка 54] Отчёт «Состояние запасов»

## 1. Карточка оценки
- Номер строки: 54
- Название: Состояние запасов (резервы, невостребованное, аварийные, страховые, ДХНО)
- Краткая формулировка требования: аналитический отчёт с разбиением запасов по управленческим категориям с увязкой резервов под потребности и нормативов.
- Дата оценки: 23.03.2026
- Версия оценки: 1.0
- Цель бизнеса: единое представление для управления уровнем запасов и перераспределения.
- Границы: отчёт и запросы к регистрам; вне scope — расчёт нормативов (11), если они ведутся вне системы.

## 2. Статус реализации
- Полностью реализовано: не подтверждено
- Частично реализовано: вероятно (типовой «Анализ/состояние запасов» может не совпадать с таксономией заказчика)
- Пояснение: категории «аварийный/страховой/ДХНО» должны быть либо вычислены из нормативов (11), либо заданы правилами; без подтверждения регистров оценка остаётся с резервом.

## 3. Подтверждённые факты
- В метаданных присутствуют объекты/подсистемы, связанные с запасами и планированием (подтверждено анализом метаданных).
- В чек-листе зафиксированы связи с 11, 40, 38.

## 4. Открытые вопросы
- Точные формулы ДХНО и границы «невостребованного» запаса.
- Источник резерва под потребность: типовой регистр обеспечения или кастом.

## 5. Новые объекты
- При разрыве: отчёт на СКД + внешние обработки выгрузки; при необходимости — вспомогательный регистр для кэша показателей.

## 6. Существующие объекты
- Регистры остатков, резервирования, обеспечения; нормативы (11); назначения (40).

## 7. Зависимости
- Внутренние: 11, 40, 38; производительность на больших объёмах.

## 8. Ролевая модель
- Чтение отчёта — широкий круг; детализация до сумм/цен может потребовать ограничений.

## 9. Варианты реализации
- **Минимально инвазивный:** доработать существующий отчёт по запасам (если есть) — добавить вычисляемые поля и отборы по категориям.
- **Архитектурный:** новый комплексный отчёт с набором вложенных схем, опциональным регламентным расчётом показателей ночью.

## 10. Класс сложности
